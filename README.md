# Windows Log Monitoring using Splunk & VirusTotal

## Overview
This project demonstrates the use of Splunk SIEM to collect and analyze Windows Application and System Event Logs for security monitoring. It integrates VirusTotal threat intelligence to identify and validate malicious file hashes, enabling real-time malware detection and SOC investigations.

## Key Features
- Collects and monitors Windows Event Logs (Application & System).  
- Integrates VirusTotal API for threat intelligence and hash validation.  
- Performs real-time log analysis and alerts generation.  
- Supports incident response and SOC operations.

## Tools & Technologies
- Splunk SIEM  
- VirusTotal API  
- Windows Event Logs  
- Security Monitoring & Threat Intelligence
